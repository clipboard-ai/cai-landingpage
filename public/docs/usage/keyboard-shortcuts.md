# Keyboard Shortcuts

Cai is designed to be used entirely from the keyboard.

## Global

| Shortcut | Action |
|---|---|
| **⌥C** | Trigger Cai (works anywhere on macOS, customizable in settings) |

## Action Window

| Shortcut | Action |
|---|---|
| **↑ ↓** | Navigate between actions |
| **↵** (Enter) | Execute selected action |
| **⌘1–9** | Jump directly to an action by number |
| **⌘0** | Open clipboard history (last 9 items) |
| **Esc** | Go back / Dismiss the window |

## Custom Prompt

| Shortcut | Action |
|---|---|
| **⌘↵** | Submit your custom prompt |
| **⌘↵** | Copy result (when viewing a result) |
| **Esc** | Back to action list |

## Menu Bar

| Action | How |
|---|---|
| Open Preferences | Left-click the Cai menu bar icon |
| Context menu | Right-click the Cai menu bar icon |
| Open Settings from action window | Click the Cai logo in the footer |
